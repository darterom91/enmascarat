System Requirements
===================
+ PrestaShop 1.6.x
+ PrestaShop 1.7.x

Configuration Instructions
==========================
    1. Upload module to your PrestaShop installation.
    2. Go to your PrestaShop administration Modules list.
    3. In module list, install Genei Module, insert user and password of Genei Module, and configure options for default shipping values.    
    4. Go to order details page and export order to Genei
